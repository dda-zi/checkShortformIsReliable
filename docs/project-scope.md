1. 프로젝트 개요
   이 시스템은 이런 문제를 해결하기 위해 사용자가 원하는 숏폼 영상의 내용을 분석하고 신뢰도를 판단하여 사용자에게 제공하는 시스템이다.
AI에 기반한 분석을 통해 영상 내용 검토 및 관련 근거 정보 제공으로 사용자가 영상의 정보를 보다 정확하게 판단 할 수 있다.

2. 배경 및 필요성
   최근 유튜브 쇼츠, 틱톡, 인스타 릴스와 같은 숏폼 컨텐츠의 소비가 급격히 증가하며 다양한 정보가 빠르게 확산되고 있다.
그러나 이런 숏폼 컨텐츠는 제작과 배포가 매우 쉽고, 짧은 시간에 소비자의 눈길을 끌어야하기에 자극적인 내용을 사용하는 경우가 많고 이로 인해 잘못된 정보, 혹은 과장된 내용이 어떠한 검증 절차 없이 무분별하게 확산되는 문제가 발생하고 있다.
이러한 잘못된 정보는 요즈음 인터넷의 엄청난 정보 확산 속도를 볼 때 사회적으로 많은 문제를 발생시킬 가능성이 높다.

3. 프로젝트 목표
  - 숏폼 영상 내용을 자동으로 분석하는 시스템 구현
  - 영상 속 주요 주장 추출
  - 관련 정보를 기반으로 신뢰도 판단
  - 사용자에게 이해하기 쉽게 분석 결과 제공

4. 시스템 범위
  입력
    - 숏폼 영상 URL 입력 or 확장 프로그램을 통한 영상 감지

  처리 과정
    - 영상에서 음성 및 텍스트 정보 추출
    - 영상 속 주요 주장 분석
    - 관련 정보 검색 및 비교

  출력
    - 영상 내용 신뢰도 점수
    - 판단 근거 정보
    - 확장 프로그램의 경우 일정 점수 이하일 시 알림
    
5. 주요 기능
   1. 숏폼 검색
     - 사용자는 숏폼의 URL을 입력한다.
     - AI가 영상의 내용을 분석해 외부 정보와 비교하고 신뢰도를 분석한다.
     - 분석된 내용을 토대로 신뢰도 점수 및 판단 근거를 보여준다.

   2. 확장 프로그램
     - 확장 프로그램의 형태로 사용할 수 있게 한다.
     - 해당 기능이 켜진 상태에서 숏폼 시청시 자동으로 영상을 감지, 판단해준다.
     - 사용자가 설정한 신뢰도 기준으로 해당 수치 이하일 시 알림으로 알려준다.
     - 판단 근거를 확인하고 싶으면 알림을 눌러서 확인할 수 있게 한다.

   3. 계정별 신뢰도 기능
      - 사람들이 신뢰도 측정을 한 숏폼 신뢰도를 DB에 저장한다.
      - DP에 저장된 내용을 토대로 숏폼 공급자의 신뢰도를 측정한다.
      - 신뢰도 랭킹 등을 만들어 공급자의 신뢰도를 파악할 수 있게 한다.
