# checkShortformIsReliable

## 프로젝트 소개
숏폼 영상의 내용을 분석해 정보의 신뢰도를 판단하고 그 정도와 근거를 제공하는 시스템이다.

## 주요 기능
- 숏폼 영상 URL 입력을 통한 검색
- 영상 내용 분석
- 신뢰도 점수 및 근거 제시
- 브라우저 확장 프로그램 지원

## 기술 스택
- Python
- FastAPI
- STT (추후 결정)
- React (Frontend)
- OCR
